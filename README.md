# Retail Sales Analysis Dashboard (2023–2025)



## Project Overview

This project analyzes retail sales performance between 2023 and 2025 using an interactive Power BI dashboard.



The goal of the analysis was to identify sales trends, product performance, and regional distribution to support business decision-making.



## Tools Used

- Microsoft Excel – Data cleaning and preparation

- Microsoft Power BI – Data visualization and dashboard creation

- DAX – KPI calculations and performance metrics



## Key KPIs

- Total Sales

- Total Orders

- Total Quantity Sold

- Sales Target Achievement



## Key Insights

- Total sales reached **541K** across three years.

- A total of **1,000 orders** were recorded with **9,746 units sold**.

- **Product D** was the top-performing product.

- The **South region** generated the highest sales.

- **March and May** recorded the highest monthly sales.
